用耳机的「播放 / 暂停」键，一键唤起语音输入，解放双手 vibe coding。
Map your headset's play/pause button to any key — fire up voice input and vibe-code hands-free.

### 📥 下载 / Downloads
- 🍎 **macOS**（Intel + Apple Silicon 通用）→ `AI接线员.dmg`
- 🪟 **Windows 10/11**（x64，免安装单文件）→ `AI接线员.exe`

### ⚠️ 首次打开会被系统拦一下，属正常现象 / First launch is blocked — this is expected
本应用未做苹果公证 / 未签名，并非软件有问题。
- **macOS**：提示「已损坏 / 身份不明」→ 系统设置 → 隐私与安全性 → **仍要打开**；或终端 `sudo xattr -cr /Applications/AI接线员.app`。另需在「隐私与安全性 → 辅助功能」里**勾选本应用**。DMG 内附图文说明。
- **Windows**：SmartScreen → **更多信息 → 仍要运行**；杀软可能因「键盘钩子」误报，请**加入白名单**。
